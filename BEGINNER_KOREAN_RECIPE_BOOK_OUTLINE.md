# 초보자용 한국 요리 레시피북 개요

## 1. 책 콘셉트

- 가제: `처음 만드는 한국 집밥`
- 대상 독자: 한식을 자주 해보지 않았지만 집에서 쉽게 따라 해보고 싶은 초보 요리자
- 핵심 약속:
  - 복잡한 기술보다 재현 가능한 기본 조리법을 먼저 익힌다.
  - 구하기 쉬운 재료와 기본 양념으로 시작한다.
  - 1~2인 가정에서 바로 써먹을 수 있는 메뉴를 중심으로 구성한다.
- 권장 방향:
  - 전체 레시피는 `준비 10분~20분`, `조리 10분~30분` 안쪽의 메뉴 위주로 편성
  - 어려운 칼질이나 장시간 육수 내기보다 `볶기`, `끓이기`, `무치기`, `부치기` 중심으로 설명

## 2. 책 구성 원칙

- 난도는 `아주 쉬움 -> 쉬움 -> 쉬움+` 순으로 올라가게 배치
- 각 레시피는 같은 형식으로 정리
  - 한 줄 소개
  - 난이도 / 조리 시간 / 분량
  - 재료
  - 양념
  - 만드는 순서
  - 실패하기 쉬운 포인트
  - 대체 가능한 재료
  - 남은 음식 활용법
- 책 앞부분에는 공통 기본기를 짧게 넣고, 본문에서는 바로 실전 레시피로 넘어간다.

## 3. 추천 목차

### Chapter 1. 시작 전에 익히는 한식 기본

- 한식 초보를 위한 기본 도구
  - 프라이팬, 냄비, 뒤집개, 칼, 도마, 계량스푼
- 자주 쓰는 기본 재료
  - 간장, 고추장, 된장, 참기름, 식용유, 다진 마늘, 고춧가루
- 밥 짓기와 보관
- 기본 양념 비율 익히기
  - 볶음 양념의 기본
  - 무침 양념의 기본
  - 찌개 간 맞추기의 기본
- 초보자가 자주 하는 실수
  - 센 불 유지로 타는 문제
  - 소금과 간장 간의 차이
  - 물 양 조절 실패

### Chapter 2. 가장 먼저 해보는 한 그릇 요리

- 계란볶음밥
- 김치볶음밥
- 참치마요덮밥
- 비빔밥
- 주먹밥

이 장의 목적:
한 번에 완성되는 메뉴로 불 조절, 볶기, 비비기 같은 가장 기본적인 동작을 익힌다.

### Chapter 3. 부담 없이 끓이는 국과 찌개

- 계란국
- 콩나물국
- 미역국
- 된장찌개
- 김치찌개

이 장의 목적:
재료 넣는 순서와 국물 간 맞추기를 익히고, 한국 가정식의 기본 국물 요리에 익숙해지게 한다.

### Chapter 4. 식탁을 채우는 쉬운 반찬

- 계란말이
- 오이무침
- 콩나물무침
- 시금치나물
- 감자조림
- 두부조림
- 어묵볶음
- 소시지야채볶음

이 장의 목적:
데치기, 무치기, 졸이기, 볶기 같은 반찬 기본기를 익히고 여러 끼에 나눠 먹는 감각을 만든다.

### Chapter 5. 주말에 해보기 좋은 전과 분식

- 김치전
- 부추전
- 감자전
- 떡볶이
- 라볶이

이 장의 목적:
반죽 농도, 팬 조리, 간식형 양념 응용을 익히면서 조금 더 재미있는 메뉴로 확장한다.

### Chapter 6. 초보자를 위한 식단 응용

- 2가지 반찬으로 차리는 가장 쉬운 집밥 한 상
- 국 1개 + 반찬 2개 조합 예시
- 남은 반찬으로 다음 끼니 이어가기
- 냉장고 기본 재료로 만드는 3일 식단 예시

이 장의 목적:
한두 가지 요리를 만드는 수준에서 끝나지 않고, 실제 식사 한 끼를 구성하는 감각까지 연결한다.

## 4. 레시피 수 기준 제안

- 권장 수록 레시피 수: 18개~23개
- 현재 초안 기준 구성:
  - 한 그릇 요리 5개
  - 국/찌개 5개
  - 반찬 8개
  - 전/분식 5개
- 가장 무난한 실속형 구성으로 줄이려면 `15개~18개` 정도로 압축해도 된다.
- 실제 출간 분량을 줄이고 싶다면 아래 12개만 먼저 선정해도 충분하다.
  - 계란볶음밥
  - 김치볶음밥
  - 계란국
  - 된장찌개
  - 김치찌개
  - 계란말이
  - 오이무침
  - 콩나물무침
  - 감자조림
  - 두부조림
  - 김치전
  - 떡볶이

## 5. 각 레시피 페이지 공통 구성안

- 제목
- 완성 사진
- 이 요리가 쉬운 이유
- 준비 시간 / 조리 시간 / 난이도 / 분량
- 재료 사진 또는 한눈에 보는 재료 표
- 순서별 조리 과정
- 초보 팁
  - 불 세기
  - 익힘 기준
  - 간 조절 방법
- 실패했을 때 복구법
  - 너무 짤 때
  - 너무 싱거울 때
  - 너무 질거나 마를 때
- 곁들이기 좋은 메뉴

## 6. 책 앞뒤에 넣으면 좋은 부가 페이지

### 앞부분

- 프롤로그: 왜 한식은 집에서 만들어 먹기 좋은가
- 계량 가이드
  - 1큰술, 1작은술, 1컵 기준
- 자주 나오는 재료 손질법
  - 파 썰기
  - 마늘 다지기
  - 김치 자르기
  - 두부 물기 빼기

### 뒷부분

- 자주 묻는 질문
  - 고추장이 없을 때
  - 멸치육수 없이 가능한지
  - 매운맛 조절 방법
- 초보자용 장보기 리스트
- 한식 기본 양념 보관법
- 가나다순 또는 재료별 인덱스

## 7. 편집 방향 제안

- 한 페이지에 정보가 너무 많지 않게 짧은 문장으로 설명
- 조리 순서는 사진과 함께 4단계에서 7단계 안쪽으로 정리
- 전문 용어보다 생활 언어 사용
- "약불", "중불", "센 불"의 기준을 그림이나 아이콘으로 반복 표기
- 초보자가 가장 불안해하는 지점을 눈에 띄게 표시
  - 언제 뒤집는지
  - 어느 정도 끓이면 되는지
  - 간은 언제 보는지

## 8. 추천 부제 후보

- `집에서 처음 시작하는 한식`
- `쉬운 재료로 만드는 한국 집밥`
- `초보도 완성하는 매일 한식`

## 9. 다음 작업 제안

- 우선 수록할 핵심 레시피 10개를 확정한다.
- 각 레시피의 표준 분량을 `1인분` 또는 `2인분` 기준으로 통일한다.
- 모든 레시피에 동일한 서술 템플릿을 적용한다.
- 책 샘플 챕터 1개를 먼저 완성해 톤을 고정한다.
